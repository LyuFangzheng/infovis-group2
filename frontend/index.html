    The application for identifying toxic subreddits

    Due to the frequent use of social media, people interact more and more on platforms in which they create, share, and/or exchange information and ideas. Some of this interactions may be negative and are intended to harm or at least discomfort one's audience, which is also known as trolling. In this application we will focus on a popular social media platform called reddit.com which hosts a large number of sub-communities called subreddits. Various subreddits have particular topics and people can share their opinion on the subject. As you can imagine, lots of these opinions might be negative. This application is designed for a reddit moderator to identify these toxic interactions by visualising the influences subreddits have on each other.

    Multiple visualisations will illustrate the toxic interactions. Firstly, a graph will give an overview of the connections. The user can zoom in on clusters to better see the influences of individual subreddits. The user can choose what the edges will show: intensity, positive relation or negative relation. Once the user has selected an interesting cluster where he/she would like to see more information about, a heatmap which shows for every subreddit in the cluster the extent of negativity will be visible. Also, it shows some more information based on semantics with Linguistic Inquiry and Word Count (LIWC) scores.

    Data was gathered from https://snap.stanford.edu/data/soc-RedditHyperlinks.html. In the data set, 55,863 subreddits(nodes) and 858,490 hyperlinks between subreddits (edges)are present.
