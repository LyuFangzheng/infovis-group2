    The application for identifying toxic subreddits

    How it works

    The user can zoom in on clusters to better see the influences of individual subreddits. The user can choose what the edges will show: intensity, positive relation or negative relation. Once the user has selected an interesting cluster where he/she would like to see more information about, a heatmap which shows for every subreddit in the cluster the extent of negativity will be visible. Also, it shows some more information based on semantics with Linguistic Inquiry and Word Count (LIWC) scores.
