    How to use:

    The bar chart on the right shows Linguistic Inquiry and Word Count (LIWC) scores, containing semantic information about the subreddit that is submitted in the search bar. Select a LIWC feature category to change the type of features to investigate. The heatmap on the left shows the sentiment of subreddit interactions, where each square represents the influence of the subreddit on the left (y-axis) on the subreddit on top (x-axis). Red squares represent negative influence, green square represent positive influence. Use the bottom dropdown to select the sorting order of subreddits in the heatmap.
